## 1. 게임 개요

### 1.1 기본 정보

| 항목 | 내용 |
|------|------|
| **타이틀** | 삼국지 패왕전 — Three Kingdoms: Conqueror |
| **장르** | 턴제 전략 RPG + 장수 수집 + 영토 경영 |
| **타겟 유저** | 25-45세 남성 중심, 삼국지 IP 팬, 전략 게임 마니아, 캐주얼 수집형 RPG 유저 |
| **세션 길이** | 1회 5-15분 (출퇴근 시간 최적화) |
| **오프라인** | 완전 오프라인 플레이 지원 (서버 불필요) |
| **과금 모델** | F2P + 하이브리드 수익화 (광고 + IAP) |
| **연령 등급** | 9+ (가벼운 만화 스타일 전투) |

### 1.2 게임 비전

> *"삼국지의 영웅들과 함께 천하를 통일하라!"*

삼국지 패왕전은 삼국시대의 역사적 사건을 턴제 전투로 체험하면서, 100명 이상의 장수를 수집·육성하고, 중원 대륙의 영토를 정복해 나가는 모바일 전략 RPG이다.

### 1.3 차별점 (USP)

| 차별점 | 설명 |
|--------|------|
| **완전 오프라인** | 경쟁작 대부분이 온라인 필수인 반면, 와이파이 없이도 플레이 가능 |
| **Non-P2W** | 과금 없이도 모든 콘텐츠 클리어 가능. 과금은 시간 단축 + 외형 위주 |
| **역사적 정확성** | 삼국지연의 + 정사 기반의 장수 데이터, 고증에 기반한 스토리 |
| **깊은 전투** | 단순 자동 전투가 아닌, 병종 상성 + 진형 + 스킬 타이밍의 전략적 전투 |
| **인디 감성** | 대형사의 과도한 과금 요소 없이, 순수하게 게임성에 집중 |

### 1.4 경쟁 분석

| 경쟁작 | 장점 | 약점 | 우리의 대응 |
|--------|------|------|------------|
| 삼국지 전략판 | 깊은 전략, 코에이 IP | 과금 부담, 온라인 필수, 시간 소모 큼 | 오프라인 + 캐주얼 세션 |
| 삼국지 오리진 | 대규모 국가전 | 과금 의존, 온라인 전용 | Non-P2W + 싱글 포커스 |
| 삼국지 올스타 | 화려한 그래픽 | 과도한 가챠 | 업적/스토리로 장수 획득 |
| 삼국지 전략판 PK | 시즌제 콘텐츠 | 기존 유저 우위 | 신규 진입 장벽 제거 |

### 1.5 타겟 시장 분석

**시장 규모 (2025 기준):**
- 글로벌 모바일 게임 시장: $103-106B

## 2. 핵심 게임 루프

### 2.1 코어 루프 다이어그램

```
┌─────────────────────────────────────────────┐
│                  코어 루프                    │
│                                              │
│  ┌──────┐    ┌──────┐    ┌──────┐           │
│  │ 전투  │───▶│ 보상  │───▶│ 성장  │          │
│  │(3-5분)│    │      │    │      │           │
│  └──┬───┘    └──────┘    └──┬───┘           │
│     │                       │                │
│     │    ┌──────────┐       │                │
│     └────│  편성     │◀──────┘                │
│          │(장수/장비)│                         │
│          └──────────┘                         │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│                 메타 루프                     │
│                                              │
│  전투 → 영토 점령 → 자원 획득 → 내정 발전    │
│    → 병력 증강 → 더 강한 전투 → 천하통일      │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│               수집 루프                      │
│                                              │
│  스토리 진행 → 장수 획득 → 육성 → 스킬 해금  │
│    → 시너지 발견 → 새 편성 → 더 깊은 전략    │
└─────────────────────────────────────────────┘
```

### 2.2 세션 설계

| 세션 유형 | 소요 시간 | 빈도 | 주요 활동 |
|-----------|-----------|------|-----------|
| **짧은 세션** | 3-5분 | 하루 4-6회 | 스테이지 1-2개 클리어, 일일 미션 |
| **중간 세션** | 10-15분 | 하루 2-3회 | 장수 육성, 장비 관리, 스토리 진행 |
| **긴 세션** | 20-30분 | 하루 1회 | 영토 확장, 챕터 보스전, 이벤트 |

### 2.3 리텐션 포인트

| 기간 | 리텐션 메카닉 | 목표 |
|------|--------------|------|
| **D1** | 도원결의 3형제 획득 + 첫 전투 승리 | 40%+ |
| **D3** | SSR 장수 1명 무료 획득 (뽑기 천장) | 25%+ |
| **D7** | 챕터2 입장 + 진형 시스템 해금 | 18%+ |
| **D14** | 영토 시스템 해금 + 세력 선택 | 12%+ |
| **D30** | 적벽대전 진입 + 시즌 이벤트 | 8%+ |

### 2.4 일일/주간 사이클

**일일 사이클:**
- 출석 보상 (7일 연속 시 SSR 장수 조각)
- 일일 퀘스트 3종 (전투 3회, 장수 강화 1회, 내정 1회)
- 무료 뽑기 1회
- 자원 수확 (영토에서 식량/금/철 자동 생산)

**주간 사이클:**
- 주간 보스 도전 (고난도 보스전)
- 주간 랭킹 보상 (전투력 기준)
- 주간 뽑기 보너스 (10+1 무료 1회 추가)

## 3. 장수 시스템

### 3.1 등급 시스템

| 등급 | 색상 | 획득 확률 (뽑기) | 초기 스탯 범위 | 스킬 수 | 최대 레벨 |
|------|------|-----------------|---------------|---------|-----------|
| **N (일반)** | 회색 | 50% | 100-200 | 패시브 1 | 30 |
| **R (희귀)** | 초록 | 30% | 200-350 | 액티브 1 + 패시브 1 | 40 |
| **SR (영웅)** | 파랑 | 15% | 350-500 | 액티브 1 + 패시브 2 | 50 |
| **SSR (전설)** | 보라/금 | 4.5% | 500-700 | 액티브 2 + 패시브 2 | 60 |
| **UR (신화)** | 빨강 | 0.5% | 700-900 | 액티브 2 + 패시브 3 | 70 |

> **천장 시스템:** 80회 뽑기 내 SSR 미획득 시 확정 SSR 지급 (Non-P2W 보장)

### 3.2 스탯 시스템

| 스탯 | 약칭 | 역할 | 범위 |
|------|------|------|------|
| **무력 (武力)** | ATK | 물리 공격력, 일기토 데미지 | 1-100 |
| **지력 (智力)** | INT | 책략 공격력, 계략 성공률 | 1-100 |
| **통솔 (統率)** | CMD | 부대 HP, 방어력 | 1-100 |
| **속도 (速度)** | SPD | 행동 순서, 회피율 | 1-100 |
| **HP** | HP | 체력 | 통솔 × 50 + 기본값 |

**파생 스탯:**
- 물리 공격력 = 무력 × 3 + 장비 보정 + 레벨 보정
- 책략 공격력 = 지력 × 3 + 장비 보정 + 레벨 보정
- 물리 방어 = 통솔 × 2 + 장비 보정
- 책략 방어 = 지력 × 1.5 + 통솔 × 0.5 + 장비 보정
- 회피율 = 속도 / 3 (최대 30%)
- 치명타율 = 무력 / 5 (최대 20%)

## 4. 전투 시스템

### 4.1 전투 기본 구조

| 항목 | 상세 |
|------|------|
| **전투 방식** | 턴제 (SPD 기준 행동 순서) |
| **참전 인원** | 아군 최대 5명 vs 적군 최대 5명 |
| **배치** | 전열 3자리 + 후열 2자리 |
| **턴 구조** | 속도 순으로 개별 행동 (아군·적군 교차 아님) |
| **전투 종료** | 한쪽 전멸 또는 20턴 경과 (20턴 시 HP% 비교) |
| **자동전투** | 2배속 + AI 자동 전투 (수동 전환 가능) |

### 4.2 행동 순서 결정

```
행동 순서 = 기본 속도 + 장비 보정 + 버프/디버프 + Random(-5, +5)

매 턴 시작 시:
1. 모든 생존 유닛의 행동 순서 값 계산
2. 높은 순서대로 정렬
3. 순서대로 한 명씩 행동
4. 행동: [일반 공격] / [스킬 사용] / [방어] / [아이템 사용]
```

### 4.3 병종 상성 시스템

```
       보병 (步兵)
      ↗        ↘
  궁병 (弓兵) ← 기병 (騎兵)

  보병 > 궁병: 데미지 +30%
  궁병 > 기병: 데미지 +30%  
  기병 > 보병: 데미지 +30%
  역상성: 데미지 -20%
```

| 공격 → 방어 | 보병 | 궁병 | 기병 |
|:----------:|:----:|:----:|:----:|
| **보병** | ×1.0 | ×1.3 | ×0.8 |
| **궁병** | ×0.8 | ×1.0 | ×1.3 |
| **기병** | ×1.3 | ×0.8 | ×1.0 |

**병종 특수 능력:**
- **보병:** 방어 명령 시 데미지 감소 50% (타 병종은 30%)
- **궁병:** 후열에서도 전열 공격 가능 (데미지 감소 없음)
- **기병:** 첫 턴에 선제 공격 시 데미지 +20% (돌격 보너스)

### 4.5 데미지 공식

#### 물리 데미지

```
기본 데미지 = (공격자 무력 × 3 + 무기 공격력) × 스킬 배율
방어 감소 = 방어자 (통솔 × 2 + 갑옷 방어력) / (통솔 × 2 + 갑옷 방어력 + 300)
상성 보정 = 1.3 (유리) / 1.0 (동등) / 0.8 (불리)
치명타 = Random < 무력/5 ? 1.5 : 1.0
최종 데미지 = 기본 데미지 × (1 - 방어 감소) × 상성 보정 × 치명타 × Random(0.9, 1.1)
```

#### 책략 데미지

```
기본 데미지 = (공격자 지력 × 3 + 장비 보정) × 스킬 배율
책략 방어 감소 = (방어자 지력 × 1.5 + 통솔 × 0.5) / (방어자 지력 × 1.5 + 통솔 × 0.5 + 250)
최종 데미지 = 기본 데미지 × (1 - 책략 방어 감소) × Random(0.9, 1.1)
```

#### HP 계산

```
최대 HP = 통솔 × 50 + 장비 HP + 레벨 × 20 + 기본값(500)
```

### 4.7 전투 AI (적군)

적 AI는 난이도별로 다른 행동 패턴을 사용:

| AI 유형 | 난이도 | 행동 패턴 |
|---------|--------|-----------|
| **무모형** | 쉬움 | 가장 가까운 적 공격, 스킬 랜덤 사용 |
| **균형형** | 보통 | HP 낮은 적 우선, 상성 고려 |
| **전략형** | 어려움 | 상성 + 스킬 효율 계산, 약한 유닛 집중 공격 |
| **두뇌형** | 극난 | 진형 파괴 우선, 힐러/책사 먼저 제거, 스킬 콤보 사용 |

**AI 판단 로직 (전략형 이상):**
```
1. 타겟 선정:
   - 상성 유리한 적 우선 (가중치 ×2)
   - HP 비율 낮은 적 우선 (가중치 × (1 - HP비율))
   - 후열 책사/궁병 우선 (가중치 ×1.5)
   
2. 스킬 선택:
   - 가용 스킬 중 예상 데미지 최대인 스킬 선택
   - HP 30% 이하 시 방어/회복 우선
   - 아군 3명+ 생존 시 버프 스킬 고려

3. 특수 조건:
   - 보스: 3턴마다 궁극기 사용
   - 보스 HP 30% 이하: 광폭화 (공격력 +50%)
```

### 7.2 재화 밸런스 설계

#### 일일 수입 (무과금 유저, 중반부 기준)

| 재화 | 전투 보상 | 생산 | 퀘스트 | 합계/일 |

| 보석 | 30 | 0 | 50 | 80 |
| 식량 | 2,000 | 2,000 (성 1개) | 1,000 | 5,000 |
| 병력 | 300 | 300 (성 1개) | 200 | 800 |
| 철광 | 500 | 300 | 200 | 1,000 |

#### 일일 지출 (중반부 기준)

| 항목 | 골드 | 식량 | 병력 | 철광 |
|------|------|------|------|------|

| **합계** | **5,500** | **2,500** | **500** | **500** |

> **설계 원칙:** 무과금 유저도 매일 약간의 잉여 재화가 남도록 설계. 과금은 진행 속도를 2-3배 가속.

#### 뽑기 경제

| 뽑기 유형 | 비용 | 일일 무료 횟수 | 천장 |
|-----------|------|---------------|------|
| 일반 뽑기 | 3,000 골드 | 1회 | 없음 (N~R만) |
| 고급 뽑기 | 300 보석 | 0회 (주 1회) | 80회 = SSR 확정 |
| 10연차 | 2,700 보석 | 0회 | 10회당 SR+ 1명 확정 |

**보석 소요 분석:**
- SSR 천장까지: 80 × 300 = 24,000 보석
- 무과금 일일 보석 수입: 80개
- 무과금 SSR 천장 도달: 약 300일 (= 10개월)
- 스토리/업적 보석 포함 시: 약 60-90일 단축 → **실질 7-8개월**
- 과금 시: $9.99 패키지 = 980 보석 → 약 25회 과금으로 천장
