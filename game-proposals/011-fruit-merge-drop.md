# 011 - Fruit Merge Drop (과일 합체 드롭)

- **"반복을 통한 깊이"** — 2026 하이퍼캐주얼 핵심 전략

## 게임 컨셉
물리 엔진 기반 과일 드롭 머지 게임. 같은 과일끼리 닿으면 한 단계 큰 과일로 합체. 
컨테이너에서 과일이 넘치면 게임 오버. 최고 점수를 노리는 무한 모드.

## 핵심 메카닉
1. **드롭**: 터치/클릭으로 x좌표 지정 → 과일이 떨어짐
2. **머지**: 같은 종류 과일 2개가 접촉 → 다음 단계 과일로 합체
3. **체인 리액션**: 합체 후 새 과일이 또 다른 같은 과일과 접촉 → 연쇄 합체
4. **스코어링**: 높은 단계 과일일수록 고득점, 체인 보너스
5. **게임오버**: 과일이 컨테이너 상단 라인을 넘으면 종료

## 과일 진화 체계 (11단계)
🍒 체리 → 🍇 포도 → 🍊 귤 → 🍋 레몬 → 🍎 사과 → 🍐 배 → 🍑 복숭아 → 🍍 파인애플 → 🍈 멜론 → 🍉 수박 → 👑 골든 프루츠

## 타겟 유저
- 캐주얼 퍼즐 게이머 (25-44세, 여성 61%)
- 짧은 세션 (3-10분) 선호
- 점수 경쟁 즐기는 유저

## 차별점
- **물리 기반 인터랙션**: 과일이 굴러가고 밀리는 물리 시뮬레이션
- **체인 리액션 쾌감**: 연쇄 합체의 도파민 자극
- **비주얼 피드백**: 합체 시 파티클 이펙트 + 화면 흔들림
- **넥스트 프리뷰**: 다음에 나올 과일 미리보기

## 수익 모델
- 광고: 게임오버 시 리워드 광고 (이어하기)
- IAP: 스킨팩 (과일 → 보석, 행성, 동물 등)
- 광고 제거 $2.99

## 기술 스택
- **HTML5 Canvas** + **JavaScript**
- 물리 엔진: 자체 구현 (원형 충돌 + 중력)
- 터치/마우스 입력
- LocalStorage 하이스코어

## 시스템 설계

### 물리 엔진
- 원형 Rigid Body (각 과일)
- 중력 가속도: 0.5px/frame²
- 충돌 감지: 원-원 거리 기반
- 충돌 응답: 탄성 계수 0.3, 마찰 0.8
- 벽/바닥 바운스

### 머지 시스템
- 같은 타입 과일 충돌 → 합체 트리거
- 합체 위치: 두 과일 중심의 중간점
- 합체 후 0.1초 쿨다운 (즉시 체인 방지)
- 최고 단계(골든 프루츠) 도달 시 특수 이펙트 + 보너스 점수

### 스코어 시스템
| 과일 단계 | 합체 점수 |
|-----------|-----------|
| 체리→포도 | 1 |
| 포도→귤 | 3 |
| 귤→레몬 | 6 |
| 레몬→사과 | 10 |
| 사과→배 | 15 |
| 배→복숭아 | 21 |
| 복숭아→파인 | 28 |
| 파인→멜론 | 36 |
| 멜론→수박 | 45 |
| 수박→골든 | 55 |

체인 보너스: 연속 합체 시 x1.5, x2.0, x3.0 ...

### UI/UX 플로우
1. **타이틀 화면**: 로고 + "탭하여 시작" + 하이스코어
2. **게임 화면**: 
   - 상단: 점수 + 하이스코어
   - 중앙: 게임 컨테이너 (과일이 쌓이는 공간)
   - 상단 우측: 다음 과일 미리보기
   - 위험선: 빨간 점선 (이 위로 과일이 올라오면 경고)
3. **게임오버**: 최종 점수 + 하이스코어 갱신 + 재시작 버튼

### 밸런스 파라미터
- 컨테이너 너비: 350px, 높이: 500px
- 과일 크기: 체리 20px ~ 수박 120px (단계별 1.3배)
- 드롭 과일: 1~5단계 랜덤 (가중치: 높은 단계일수록 낮은 확률)
- 위험선: 컨테이너 상단에서 50px 아래
- 게임오버 판정: 위험선 위에 2초 이상 과일 존재
