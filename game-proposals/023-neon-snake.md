# Neon Snake — 네온 스네이크

## 게임 컨셉
클래식 스네이크에 네온 비주얼과 파워업 시스템을 결합한 모던 아케이드 게임. 간단한 조작으로 중독성 있는 플레이를 제공하며, 파워업과 콤보 시스템으로 깊이를 추가.

## 핵심 메카닉
- **스네이크 이동**: 스와이프(모바일) / 방향키·WASD(PC)로 뱀 조종
- **먹이 시스템**: 일반 먹이 + 특수 먹이 (보너스, 스피드, 고스트, 마그넷)
- **콤보 시스템**: 빠른 연속 먹이 섭취 시 점수 배율 상승
- **난이도 상승**: 길이가 길어질수록 속도 증가
- **네온 비주얼**: 발광 효과, 파티클, 그라디언트 뱀 몸체

## 타겟 유저
- 클래식 게임 향수를 가진 유저
- 캐주얼 아케이드 선호 유저
- 하이스코어 경쟁을 좋아하는 유저

## 차별점
1. **네온 아트 스타일**: 어두운 배경에 빛나는 뱀과 먹이
2. **파워업 다양성**: 4종 특수 먹이로 전략적 플레이
3. **콤보 메카닉**: 빠른 먹이 연속 섭취로 스코어 폭발
4. **반응형 컨트롤**: 터치 스와이프 + 키보드 완벽 지원
5. **파티클 이펙트**: 먹이 섭취/사망 시 화려한 효과

## 기술 스택
- HTML5 Canvas (60fps 렌더링)
- Vanilla JavaScript
- LocalStorage (하이스코어)
- requestAnimationFrame 기반 게임 루프

## 시스템 설계

### 게임 상태
```javascript
{
  snake: [{x, y}],      // 뱀 세그먼트 배열
  direction: {x, y},     // 현재 방향
  food: {x, y, type},    // 현재 먹이
  score: number,
  highScore: number,
  combo: number,
  speed: number,          // ms per move
  powerUp: string|null,   // 활성 파워업
  gameState: 'menu'|'playing'|'dead'
}
```

### 먹이 타입 (4종)
1. 🍎 일반 (10점, 80% 확률)
2. ⭐ 보너스 (50점, 10% 확률)
3. ⚡ 스피드 (20점, 5% 확률, 일시적 속도 증가)
4. 👻 고스트 (30점, 5% 확률, 일시적 벽 통과)

### 점수 계산
- 기본: 먹이 포인트 × (1 + combo × 0.5)
- 콤보: 3초 이내 연속 먹이 섭취 시 콤보 +1
- 최대 콤보 배율: ×5.0

## 밸런스 파라미터
- 그리드: 20×20
- 시작 속도: 150ms/이동
- 최소 속도: 60ms/이동
- 속도 감소: 먹이당 2ms
- 콤보 타임아웃: 3초
- 고스트 지속: 5초
- 스피드 부스트: 3초
