# 007 - Gravity Orbit (중력 궤도)

## 게임 컨셉
우주 공간에서 행성 사이를 궤도 이동하며 별을 수집하는 물리 기반 캐주얼 게임.
플레이어는 위성을 조작하여 행성 주위를 공전하다가, 탭하면 궤도에서 이탈해 다음 행성의 중력장에 진입한다.
체인 궤도를 연결하며 우주를 탐험하는 만족감 있는 게임플레이.

## 핵심 메카닉
- **원터치 궤도 이탈**: 탭하면 현재 궤도에서 접선 방향으로 이탈
- **중력 캡처**: 다른 행성 근처에 가면 자동으로 궤도 진입
- **체인 보너스**: 연속 궤도 전환 시 점수 배율 증가
- **장애물 회피**: 소행성, 블랙홀 등 위험 요소
- **별 수집**: 궤도 경로에 있는 별을 수집하여 점수 획득

## 타겟 유저
- 캐주얼 게이머 (15-45세)
- 짧은 세션 선호 (1-3분)
- 시각적 만족감을 중시하는 유저
- 물리 기반 게임 팬

## 차별점
- 궤도 역학 기반의 독특한 조작감
- 네온 우주 비주얼 + 궤적 트레일 효과
- 단순 조작이지만 깊은 타이밍 마스터리
- 절차적 레벨 생성으로 무한 리플레이

## 수익 모델
- 광고 기반 (인터스티셜, 리워드 영상)
- 스킨 판매 (위성, 궤적 이펙트)
- 부활 아이템

## 기술 스택
- HTML5 Canvas 2D
- 바닐라 JavaScript (외부 라이브러리 없음)
- requestAnimationFrame 기반 게임 루프
- 터치/마우스 이벤트 통합

## 시스템 설계

### 물리 시스템
- 행성: 위치(x,y), 반지름, 중력 범위, 중력 강도
- 위성: 위치, 속도 벡터, 상태(궤도/자유비행)
- 궤도 상태: 각속도 계산 (중력 = mv²/r)
- 자유비행: 등속 직선 운동 + 근처 행성 중력 감지
- 충돌: 행성 표면 충돌 = 게임오버, 소행성 충돌 = 게임오버

### 레벨 생성
- 카메라: 위성 중심 추적
- 행성 배치: 현재 위치 기준 전방에 랜덤 생성
- 난이도 곡선: 거리 증가 → 행성 간격 넓어짐, 소행성 증가
- 무한 스크롤 방식

### 점수 시스템
- 별 수집: 100점/개
- 궤도 체인: 1x → 2x → 3x ... (미스 시 리셋)
- 거리 보너스: 이동 거리 비례

## UI/UX 플로우

### 메인 메뉴
- 게임 타이틀 + 우주 배경 애니메이션
- "탭하여 시작" 버튼
- 최고 점수 표시

### 게임플레이
- 위성이 첫 행성 궤도에서 시작
- 화면 탭 → 궤도 이탈
- 다음 행성 근처 도달 → 자동 궤도 진입
- HUD: 점수, 체인 배율, 수집한 별
- 궤적 트레일 이펙트 (네온 글로우)

### 게임오버
- 행성/소행성 충돌 또는 화면 밖 이탈
- 최종 점수 + 최고 점수
- "다시 시작" 버튼

## 밸런스 파라미터
```
행성 반지름: 30-80px
궤도 반지름: 행성 반지름 + 40-80px
궤도 각속도: 0.02-0.05 rad/frame
행성 간 거리: 200-400px (난이도에 따라 증가)
중력 캡처 범위: 궤도 반지름 + 30px
소행성 등장: 점수 500점부터
소행성 속도: 1-3px/frame
별 출현 빈도: 행성당 1-3개
체인 배율 상한: 10x
```

## 비주얼 스타일
- 다크 우주 배경 + 별 파티클
- 네온 글로우 행성 (다양한 색상)
- 위성 궤적: 점점 사라지는 네온 트레일
- 별 수집 시 파티클 이펙트
- 체인 보너스 시 화면 플래시
