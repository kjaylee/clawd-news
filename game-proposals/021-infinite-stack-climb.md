# 021. Infinite Stack Climb

## 게임 컨셉
하늘 높이 쌓아 올라가는 스택 빌딩 게임. 클래식 Stack 메카닉에 리텐션 요소(퀘스트, 스킨, 랭킹)를 추가한 2026년형 하이퍼캐주얼.

## 핵심 메카닉
- **원탭 스택**: 타이밍에 맞춰 탭하면 블록이 쌓임
- **완벽 스택 보너스**: 정확히 맞추면 콤보 + 블록 확장
- **고도 기반 난이도**: 높이 올라갈수록 속도 증가
- **환경 변화**: 낮→저녁→밤→우주 배경 전환
- **파워업**: 자석(자동 정렬), 슬로우, 2배 점수

## 타겟 유저
- 캐주얼 게이머 (10-40대)
- 짧은 플레이 세션 선호 (2-5분)
- 인플루언서 콘텐츠 시청자

## 차별점
1. **리텐션 시스템**: 일일 퀘스트, 주간 챌린지
2. **커스터마이징**: 블록 스킨, 배경 테마 수집
3. **소셜**: 글로벌 랭킹, 친구 대결
4. **스트리머 친화적**: 깔끔한 비주얼, 리액션 유발

## 수익 모델
- **광고**: 리워드 광고 (부활, 2배 코인)
- **IAP**: 광고 제거 $2.99, 스킨 팩 $0.99-$1.99
- **배틀패스**: 시즌 보상 $4.99

## 기술 스택
- HTML5 + Canvas API
- 로컬 스토리지 (진행 저장)
- CSS 애니메이션 + requestAnimationFrame

## 상세 설계

### 시스템 설계
```
GameManager
├── StackController (블록 이동/배치)
├── ScoreSystem (점수/콤보/보너스)
├── ProgressionManager (레벨/퀘스트)
├── SkinManager (스킨/테마)
└── AudioManager (효과음/배경음)
```

### UI/UX 플로우
1. **메인 메뉴**: Play, Shop, Quests, Leaderboard
2. **게임 플레이**: 
   - 상단: 점수, 콤보, 고도 표시
   - 중앙: 스택 영역
   - 하단: 파워업 슬롯
3. **게임 오버**: 점수, 최고 기록, 부활/재시작

### 밸런스 파라미터
```javascript
const BALANCE = {
  initialSpeed: 3,        // 초기 블록 속도
  speedIncrement: 0.15,   // 층당 속도 증가
  maxSpeed: 12,           // 최대 속도
  perfectTolerance: 5,    // 완벽 판정 허용 픽셀
  comboMultiplier: 0.1,   // 콤보당 점수 배율 증가
  skinUnlockHeight: [50, 100, 200, 500, 1000],
  questRewards: {
    daily: { coins: 50, xp: 100 },
    weekly: { coins: 200, xp: 500, skin: true }
  }
};
```

### 구현 우선순위
1. 기본 스택 메카닉 (필수)
2. 점수/콤보 시스템
3. 환경 변화 (높이별 배경)
4. 파워업 시스템
5. 퀘스트/진행도 (MVP 이후)
