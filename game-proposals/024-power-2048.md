# 024 — Power 2048

## 📋 게임 컨셉
**2048 변형 퍼즐** — 클래식 2048 메카닉에 파워업 시스템과 진행 요소를 결합한 하이브리드 캐주얼 게임.

> **트렌드 근거:** "Stack and 2048 gained massive traction as influencers and casual streamers featured them in bite-sized content" — 숏폼 바이럴 최적화 게임. 2026년 "Depth through Repetition" 원칙에 완벽 부합.

## 🎮 핵심 메카닉

### 기본 플레이
- 4×4 그리드, 스와이프로 타일 이동
- 같은 숫자 타일이 충돌하면 합쳐져 2배 값
- 목표: 2048 타일 달성 (이후 무한 모드)

### 🔥 파워업 시스템 (차별점)
1. **💣 Bomb (폭탄)** — 선택한 타일 + 주변 8칸 제거. 막힌 보드 구출용
2. **❄️ Freeze (동결)** — 한 타일 고정. 3턴간 합쳐지지 않음. 전략적 배치용
3. **⚡ Double (더블)** — 다음 합칠 때 4배 값. 빠른 성장
4. **🔄 Undo (되돌리기)** — 마지막 1수 취소
5. **🎯 Sniper (스나이퍼)** — 특정 타일 하나만 제거

### 파워업 획득
- 합치기 콤보 (3연속 합치기 시 랜덤 파워업)
- 스테이지 클리어 보상
- 리워드 광고 시청

## 🎯 타겟 유저
- **1차:** 25-44세 여성 캐주얼 게이머 (시장 61%)
- **2차:** 퍼즐 좋아하는 전연령층
- **3차:** 인플루언서/스트리머 (숏폼 콘텐츠 적합)

## 🏆 차별점
1. **파워업 시스템** — 기존 2048 클론에 없는 전략적 깊이
2. **스테이지 모드** — 목표가 있는 레벨 진행 (클래식 2048은 끝이 없음)
3. **일일 챌린지** — 매일 고유 보드 + 글로벌 리더보드
4. **비주얼 테마** — 네온, 우주, 자연 등 테마 언락

## 💰 수익 모델
1. **리워드 광고 (메인)** — 파워업 획득, 추가 생명
2. **배너 광고** — 하단 배너 (스테이지 간)
3. **IAP:**
   - 파워업 팩 ($0.99 ~ $4.99)
   - 광고 제거 ($3.99)
   - 테마 팩 ($1.99)
4. **구독 (VIP):** $2.99/월 — 광고 제거 + 매일 파워업 + 전용 테마

## 🛠 기술 스택
- **엔진:** Vanilla HTML5 Canvas + JavaScript
- **렌더링:** 2D Canvas API
- **애니메이션:** requestAnimationFrame + CSS transitions
- **저장:** localStorage
- **배포:** GitHub Pages (정적)


### 게임 모드
1. **Classic** — 무한 모드. 최고 점수 도전
2. **Stage** — 100+ 레벨. 각 레벨 목표 점수 달성
3. **Daily Challenge** — 매일 새 보드. 시드 기반 (모든 유저 동일)

### 스테이지 레벨 설계
| 레벨 | 목표 | 특수 조건 | 파워업 |
|------|------|-----------|--------|
| 1-10 | 256 달성 | 없음 | Undo ×2 |
| 11-20 | 512 달성 | 30수 제한 | Bomb ×1 |
| 21-30 | 1024 달성 | 돌 타일 1개 | Double ×1 |
| 31-50 | 2048 달성 | 돌 타일 2개 + 25수 | 풀 세트 |
| 51+ | 4096+ | 혼합 조건 | 랜덤 |

### 돌 타일 (장애물)
- 합쳐지지 않는 고정 타일
- 폭탄으로만 제거 가능
- 스테이지 난이도 조절 핵심
