# Merge Rush - 머지 러시

## 게임 컨셉
2048 스타일 머지 + 하이브리드캐주얼 요소. 같은 숫자를 합쳐 더 큰 숫자를 만들되, **경쟁 요소**와 **진행 시스템**을 추가하여 리텐션 강화.

## 핵심 메카닉
- **스와이프 머지**: 상하좌우 스와이프로 블록 이동, 같은 숫자 충돌 시 합체
- **시간 제한 모드**: 60초 내 최고 점수 달성 (경쟁 요소)
- **일일 챌린지**: 매일 같은 시드로 전 세계 유저와 경쟁
- **코인 시스템**: 플레이 시 코인 획득 → 스킨/파워업 구매

## 트렌드 반영
- **하이브리드캐주얼**: 단순한 코어 + 메타 진행 시스템
- **경쟁 요소**: 일일 랭킹, 시간 제한 모드
- **리텐션 강화**: 일일 보상, 업적, 컬렉션

## 타겟 유저
- 2048 팬층
- 경쟁 좋아하는 캐주얼 게이머
- 짧은 세션 (1-3분) 선호 유저

## 차별점
1. **시간 제한 모드**: 긴장감 + 리플레이 가치
2. **일일 챌린지**: 동일 조건 글로벌 경쟁
3. **스킨 시스템**: 숫자 블록 커스터마이징
4. **파워업**: 폭탄 (3x3 클리어), 셔플, 되돌리기

## 수익 모델
- **리워드 광고**:
  - 추가 시간 +30초
  - 파워업 1회 사용
  - 코인 2배
- **IAP**:
  - 광고 제거: $2.99
  - 코인 팩: $0.99 ~ $9.99
  - 스킨 팩: $1.99

## 기술 스택
- HTML5 Canvas
- Vanilla JavaScript
- LocalStorage (진행/랭킹)
- PWA 지원

## 시스템 설계

### 게임 상태
```javascript
{
  grid: [4][4],           // 2의 거듭제곱 값 또는 0
  score: number,
  highScore: number,
  timeLeft: number,       // 시간 제한 모드용
  coins: number,
  powerUps: {
    bomb: number,
    shuffle: number,
    undo: number
  },
  dailySeed: string,      // 일일 챌린지용
  unlockedSkins: [],
  currentSkin: string
}
```

### 게임 모드
1. **클래식**: 시간 제한 없음, 게임 오버까지
2. **러시**: 60초 제한, 최고 점수 도전
3. **일일 챌린지**: 당일 시드, 1회 도전, 글로벌 랭킹

### 점수 계산
- 블록 합체: 합쳐진 숫자 값
- 콤보: 연속 합체 시 1.5x 배율
- 시간 보너스 (러시 모드): 남은 시간 × 10

### 파워업
| 파워업 | 효과 | 가격 (코인) |
|--------|------|-------------|
| 💣 폭탄 | 선택한 3x3 영역 클리어 | 500 |
| 🔀 셔플 | 블록 위치 랜덤 재배치 | 300 |
| ↩️ 되돌리기 | 마지막 이동 취소 | 200 |

## 밸런스 파라미터

### 난이도
- 그리드: 4x4
- 시작 블록: 2개 (값: 2 또는 4)
- 새 블록: 이동 후 90% 확률 2, 10% 확률 4
- 게임 오버: 이동 불가능 시

### 시간 모드
| 모드 | 시간 | 추가 시간 (광고) |
|------|------|-----------------|
| 러시 | 60초 | +30초 (1회) |
| 일일 | 90초 | 불가 |

### 코인 경제
| 행동 | 코인 |
|------|------|
| 클래식 플레이 | 점수 ÷ 100 |
| 러시 플레이 | 점수 ÷ 50 |
| 일일 챌린지 완료 | 200 + 순위 보너스 |
| 광고 시청 | 50 |

### 스킨 가격
- 기본: 무료
- 레트로: 1,000 코인
- 네온: 2,000 코인
- 골드: 5,000 코인
