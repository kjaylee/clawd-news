# 015 - Number Drop Puzzle (숫자 드롭 퍼즐)

## 게임 컨셉
2048과 Suika Game을 결합한 물리 기반 숫자 합치기 퍼즐.
숫자 공을 떨어뜨려 같은 숫자끼리 충돌 시 합쳐지며 더 큰 숫자로 진화.

## 핵심 메카닉
- **드롭 시스템**: 좌우로 이동하며 숫자 공 떨어뜨리기
- **물리 충돌**: 공들이 굴러다니며 자연스럽게 충돌
- **숫자 합치기**: 2+2=4, 4+4=8, 8+8=16... 최대 2048 목표
- **연쇄 반응**: 합쳐진 공이 다른 공과 또 합쳐지면 콤보
- **오버플로우**: 공이 상단 라인 넘으면 게임오버

## 타겟 유저
- 2048 팬층
- Suika Game (스이카 게임) 유저
- 캐주얼 퍼즐 선호 20~40대

## 차별점
- Suika Game은 과일, 이건 숫자 → 더 직관적
- 2048은 그리드 기반, 이건 물리 기반 → 신선함
- 간단한 조작으로 깊은 전략성

## 수익 모델
- 광고: 게임오버 시 리워드 광고로 컨티뉴
- 인앱 구매: 스킨 (네온, 골드 등)
- 배너 광고 (하단)

## 기술 스택
- HTML5 Canvas
- 물리 엔진: 간단한 원형 충돌 직접 구현
- 저장: localStorage

# 상세 설계

## 시스템 설계

### 물리 시스템
```
- 중력: 0.5 px/frame²
- 바운스: 0.3 (벽/바닥)
- 마찰: 0.99
- 공 반지름: 숫자에 비례 (2=20px, 4=25px, 8=30px...)
```

### 숫자 진화 체계
```
2 → 4 → 8 → 16 → 32 → 64 → 128 → 256 → 512 → 1024 → 2048
(각 단계별 색상: 빨강 → 주황 → 노랑 → 초록 → 파랑 → 남색 → 보라 → 금색)
```

### 점수 시스템
```
- 합치기: 합쳐진 숫자 × 콤보 배수
- 콤보: 1초 내 연속 합치기 시 ×2, ×3...
- 최고기록: localStorage 저장
```

| 파라미터 | 값 | 설명 |
