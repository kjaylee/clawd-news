# 012 - Crystal Match Quest

## 게임 컨셉
Match-3 + Merge 하이브리드 퍼즐 게임. 보석을 3개 이상 매칭하면 크리스탈로 합체되고, 크리스탈끼리 다시 매칭하면 더 강력한 크리스탈이 생성된다. 2026년 최대 트렌드인 "하이브리드 캐주얼"을 정면으로 공략.

## 핵심 메카닉
1. **클래식 Match-3**: 인접 보석 스와이프로 3개 이상 매칭
2. **머지 진화**: 매칭된 보석 → 크리스탈로 진화 (4개 매치 = Tier 1, 5개 = Tier 2, L/T형 = Tier 3)
3. **크리스탈 체인**: 크리스탈끼리 매칭 시 폭발 + 보드 클리어 효과
4. **콤보 시스템**: 연쇄 매칭 시 멀티플라이어 + 화면 이펙트
5. **스타 수집**: 특정 점수 달성 시 ⭐ 획득 (1-3스타 per 레벨)

## 게임 모드
- **레벨 모드**: 제한된 이동 횟수 내 목표 달성 (match X개, 특정 크리스탈 생성 등)
- **무한 모드**: 시간 제한 없이 최고 점수 도전

## 타겟 유저
- 여성 25-44세 (캐주얼 게임 61% 비중)
- Match-3 기존 유저 (Candy Crush, Royal Match 유저층)
- 머지 게임 유저 (Travel Town, Merge 계열)
- 짧은 세션 선호자 (1회 플레이 2-3분)

## 차별점
1. **Match-3 + Merge 결합**: 단순 매칭에 그치지 않고 크리스탈 진화 → 전략적 깊이
2. **비주얼 프로그레션**: 크리스탈이 Tier별로 시각적으로 화려하게 진화
3. **깊이는 반복에서 온다**: 메카닉을 추가하지 않고, 같은 메카닉의 깊이를 레벨별로 증가
4. **마이크로 챌린지**: 레벨마다 미묘하게 다른 목표 (특정 색상 클리어, 크리스탈 N개 생성 등)

## 수익 모델
- **보상형 광고**: 추가 이동 횟수, 힌트, 파워업 부스터
- **인터스티셜 광고**: 레벨 사이 (행동 기반 최적화)
- **인앱 구매**: 광고 제거, 부스터 팩, 코스메틱 보드 테마

## 기술 스택
- **HTML5 + Canvas API** (순수 JavaScript)
- **터치 + 마우스 양쪽 지원**
- 외부 라이브러리 없이 순수 구현 (경량화)

## 시스템 설계

### 보드 구조
- 8×8 그리드 (표준 match-3)
- 6종류 보석 (색상: 빨강, 파랑, 초록, 노랑, 보라, 주황)
- 크리스탈 Tier 1-3 (각 색상별)

### 매칭 로직
```
1. 스와이프 → 두 보석 위치 교환
2. 매칭 검사 (가로/세로 3+)
   - 3매치: 제거 + 점수
   - 4매치: 제거 + Tier 1 크리스탈 생성
   - 5매치: 제거 + Tier 2 크리스탈 생성
   - L/T형: 제거 + Tier 3 크리스탈 생성 (십자 폭발)
3. 크리스탈 매칭: 
   - Tier 1 × 3 = 행/열 클리어
   - Tier 2 × 3 = 3×3 범위 폭발
   - Tier 3 × 3 = 동일 색상 전체 제거
4. 빈 칸 → 위에서 새 보석 낙하
5. 연쇄 매칭 검사 (cascade)
```

### 레벨 시스템
- 레벨 1-10: 튜토리얼 (기본 매칭)
- 레벨 11-30: 크리스탈 소개 (Tier 1)
- 레벨 31-60: 고급 크리스탈 (Tier 2-3)
- 레벨 61+: 장애물 도입 (잠긴 보석, 얼음 등)
- 자동 생성 알고리즘으로 무한 레벨

### 목표 유형
- "N개의 빨간 보석 제거"
- "크리스탈 N개 생성"  
- "N점 달성"
- "장애물 N개 파괴" (후반)

## UI/UX 플로우
```
[메인 화면]
  ├── 레벨 선택 (맵 형태, 스타 표시)
  ├── 무한 모드
  └── 설정

[게임 화면]
  ├── 상단: 레벨 번호 / 목표 / 남은 이동 횟수
  ├── 중앙: 8×8 보드 (스와이프 조작)
  ├── 하단: 점수 / 콤보 표시 / 부스터
  └── 결과: 클리어 (1-3스타) / 실패 (재시도/광고)
```

## 밸런스 파라미터
- 초기 이동 횟수: 20-25회 (레벨별 조정)
- 기본 점수: 3매치 = 30점, 4매치 = 80점, 5매치 = 150점
- 크리스탈 보너스: Tier 1 = ×2, Tier 2 = ×3, Tier 3 = ×5
- 콤보 멀티플라이어: 2연쇄 = ×1.5, 3연쇄 = ×2.0, 4+ = ×2.5
- 스타 기준: ★ = 목표 달성, ★★ = 점수 150%, ★★★ = 점수 200%
- 레벨 클리어율 목표: 70-80% (첫 시도)

## 시장 근거 (2026 리서치)
- Match-3 시장: $9.7B (캐주얼 전체의 40%)
- 하이브리드 캐주얼 퍼즐 매출 430% 성장 (2025)
- Merge + Match-3 결합 시 LTV 21% 증가
- 핵심 타겟 여성 25-44세가 퍼즐 지출의 56%
- Royal Match 이벤트 빈도 대비 높은 D7 리텐션 21% 향상
