# 017 - Slide Block Match (슬라이드 블록 매치)

## 게임 컨셉
8x8 그리드에서 블록을 한 방향으로 밀어 3개 이상 같은 색을 연결하면 폭발하는 퍼즐 게임.
기존 매치-3 게임과 달리 **스와이프 방향으로 블록 전체가 이동**하는 차별화된 메카닉.

## 핵심 메카닉
- **슬라이드**: 블록을 탭하고 상하좌우로 스와이프하면 해당 방향 끝까지 이동
- **매치**: 3개 이상 같은 색이 연결되면 자동 폭발
- **연쇄**: 폭발 후 빈 공간에 위 블록이 떨어지며 추가 매치 발생
- **특수 블록**: 4매치 = 라인 폭탄, 5매치 = 전체 색상 제거

## 타겟 유저
- 캐주얼 퍼즐 게임 유저
- 20-40대 남녀
- 짧은 플레이 타임 선호 (1판 30초~2분)

## 차별점
1. **능동적 조작**: 기존 매치-3는 인접 교환, 이 게임은 자유 슬라이드
2. **전략성**: 블록 이동 경로 계산 필요
3. **속도감**: 빠른 연쇄 반응으로 시원한 쾌감

## 수익 모델
- 광고: 게임 오버 시 리워드 광고로 Continue
- IAP: 광고 제거, 특수 아이템 팩

## 기술 스택
- HTML5 Canvas
- Vanilla JavaScript
- CSS3 애니메이션

---

## 시스템 설계

### 게임 모드
1. **타임 어택**: 60초 내 최대 점수
2. **무한 모드**: 더 이상 매치 불가능할 때까지

### 점수 시스템
- 기본 매치: 100점 × 블록 수
- 연쇄 보너스: 연쇄 수 × 1.5배
- 특수 블록 사용: 2배

### 블록 종류
- 기본 블록: 빨강, 파랑, 초록, 노랑, 보라 (5색)
- 라인 폭탄: 가로 또는 세로 한 줄 폭발
- 컬러 폭탄: 해당 색상 전체 제거

## 밸런스 파라미터

| 파라미터 | 값 | 설명 |
|----------|-----|------|
| GRID_SIZE | 8 | 8x8 그리드 |
| COLORS | 5 | 블록 색상 수 |
| MATCH_MIN | 3 | 최소 매치 개수 |
| TIME_LIMIT | 60 | 타임어택 시간(초) |
| LINE_BOMB_THRESHOLD | 4 | 라인폭탄 생성 조건 |
| COLOR_BOMB_THRESHOLD | 5 | 컬러폭탄 생성 조건 |
| BASE_SCORE | 100 | 블록당 기본 점수 |
| CHAIN_MULTIPLIER | 1.5 | 연쇄 배율 |
