# 006 - Hex Drop Puzzle

## 게임 컨셉
헥사곤 그리드 위에 다양한 모양의 헥스 블록을 배치하여 줄을 완성하는 퍼즐 게임.
"테트리스 + 헥사곤"의 조합으로, 6방향 라인 클리어의 만족감을 제공한다.

## 핵심 메카닉
- **드래그 앤 드롭**: 하단 트레이에서 헥스 피스를 그리드로 드래그
- **6방향 라인 클리어**: 3개 축(↗↘→) 중 어느 방향이든 한 줄이 완성되면 클리어
- **3피스 트레이**: 한번에 3개 피스가 제공, 모두 배치해야 새 피스 등장
- **게임 오버**: 남은 피스를 놓을 공간이 없으면 종료
- **콤보 시스템**: 한 번에 여러 줄 클리어 시 콤보 점수 보너스

## 타겟 유저
- 캐주얼 퍼즐 게임 유저 (25-45세)
- 1010!, Block Blast, Hex FRVR 플레이어
- 짧은 세션(2-5분) 반복 플레이 선호

## 차별점
- **시각적 만족감**: 헥사곤 클리어 시 벌집 붕괴 + 파티클 이펙트
- **전략적 깊이**: 6방향 배치로 사각 그리드 대비 전략 다양성 증가
- **미니멀 디자인**: 네온 색상 + 다크 배경의 세련된 비주얼
- **무한 모드**: 시간 제한 없이 최고 점수 도전

## 수익 모델
- 광고 기반 (배너 + 리워드 동영상)
- IAP: 테마 팩, 피스 스킨
- 리워드 광고로 "되돌리기" 기능

## 기술 스택
- **HTML5 Canvas** (순수 JavaScript)
- 모바일 터치 + 데스크톱 마우스 지원
- LocalStorage로 최고 점수 저장
- 60fps 애니메이션

## 시스템 설계

### 그리드 구조
- 정육각형 그리드, 반지름 5 (중심 포함 총 61셀)
- Axial 좌표계 (q, r) 사용
- 각 셀: {q, r, color, occupied}

### 피스 종류 (7종)
1. **싱글**: 1칸
2. **듀오**: 2칸 직선
3. **트리오 직선**: 3칸 직선
4. **트리오 꺾임**: 3칸 L형
5. **쿼드 직선**: 4칸 직선
6. **쿼드 Z형**: 4칸 Z형
7. **트라이앵글**: 3칸 삼각형

### 라인 클리어 로직
- 3개 축 방향으로 검사
- 한 줄의 모든 셀이 occupied → 클리어
- 동시 다중 클리어 → 콤보 (x1.5, x2, x3...)

### 색상 팔레트 (네온)
- #FF6B6B (레드)
- #4ECDC4 (틸)
- #45B7D1 (블루)
- #96CEB4 (민트)
- #FFEAA7 (옐로)
- #DDA0DD (퍼플)

### UI/UX 플로우
1. **타이틀 화면**: 로고 + "TAP TO START" + 최고 점수
2. **게임 화면**: 상단 점수 | 중앙 그리드 | 하단 피스 트레이
3. **게임 오버**: 점수 + 최고 점수 + "다시 하기" 버튼
4. **피스 드래그**: 반투명 프리뷰 표시, 배치 가능하면 초록 하이라이트

### 밸런스 파라미터
- 기본 점수: 셀당 10점
- 라인 클리어 보너스: 100점 × 라인수
- 콤보 배율: 1줄=x1, 2줄=x1.5, 3줄+=x2
- 피스 색상: 랜덤 6색 중 선택
- 피스 종류: 균등 랜덤 (크기 가중치 없음)
