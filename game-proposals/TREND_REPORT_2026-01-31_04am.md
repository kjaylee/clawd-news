# 트렌드 리포트 2026-01-31 04:00 AM

## 🔥 주요 트렌드

### 1. 하이브리드 캐주얼의 지배
- 하이퍼캐주얼의 시대는 쇠퇴, **하이브리드 캐주얼**이 대세
- 핵심: 간단한 시작 → 점진적 깊이 추가 → 배틀패스/시즌 이벤트
- 예시: Voodoo의 Mob Control, Rollic의 블록 슬라이더
- **허니문 기간** (광고 없는 초반) → 점진적 수익화

### 2. 머지 + 미드코어 퓨전
- Merge Vikings, Merge Stories — 머지 메카닉 + 미드코어 깊이
- Fiona's Farm — 매치3 + 타이쿤 + 어드벤처 결합
- 단순 머지를 넘어 다른 장르와의 **교차 결합** 트렌드

### 3. 아이들 + 액션 하이브리드
- RPG/타워디펜스/RTS와 아이들 메카닉 결합
- 자동 진행 + 전략적 개입 = 짧은 세션 + 장기 유지

### 4. 데일리 퍼즐 챌린지
- Puzzmo 앱 출시 → 데일리 퍼즐 모델 인기
- Wordle 영향 지속 — 하루 한 번, 공유 가능한 결과
- "just one more level" 루프 + 일일 이벤트 조합

### 5. 블록 슬라이더의 진화
- 단순 블록 퍼즐 → 점진적 메카닉 추가
- 부스터(시간 연장, 폭탄, 실패 팩) + 광고 보상
- 시즌 이벤트 + 배틀패스로 리텐션 강화

### 6. 퍼즐+워드 게임 강세 지속
- Wordscapes, Candy Crush Saga — 젊은층+시니어층 모두 타겟
- 보상적 진행감, 짧은 레벨, 지속적 챌린지

## 📊 시사점 (East Sea Games)
- 기존 42+개 게임 중 **하이브리드화** 가능한 게임 선별 필요
- 배틀패스/데일리챌린지 시스템 공통 모듈화 고려
- 머지+X 퓨전 신작 기획 가치 높음
- 블록/퍼즐 게임에 점진적 깊이 레이어 추가
