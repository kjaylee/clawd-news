# 🎮 게임 디자인 체크리스트

> 구현 전 반드시 확인할 것

## 1. 퍼즐 검증 (필수)
- [ ] **실패 조건이 있는가?** (게임오버 가능?)
- [ ] **잘못된 선택이 존재하는가?** (아무거나 눌러도 되면 ❌)
- [ ] **순서/전략이 필요한가?** (생각 없이 클리어 가능하면 ❌)
- [ ] **난이도 곡선이 있는가?** (레벨 1과 레벨 10이 같으면 ❌)

## 2. 원본 게임 규칙 분석
- [ ] 실제 인기 게임 플레이해봤는가?
- [ ] 핵심 메카닉이 무엇인지 파악했는가?
- [ ] 단순화하면서 빠진 요소가 없는가?

## 3. "바보 테스트"
구현 후 다음 방식으로 플레이:
- [ ] **랜덤 클릭** → 클리어 되면 실패
- [ ] **첫 번째 것만 계속 클릭** → 클리어 되면 실패
- [ ] **생각 없이 순서대로** → 클리어 되면 실패

## 4. 게임 유형별 필수 요소

### 정렬 퍼즐 (Ball Sort, Screw Sort 류)
- 이동 제약 (맨 위만, 같은 색 위에만 등)
- 빈 공간 제한
- 막힐 수 있는 상황 존재

### 매칭 퍼즐 (Match 3, Mahjong 류)
- 접근 제한 (위에 덮인 것 못 꺼냄)
- 슬롯 제한 (가득 차면 게임오버)
- 매칭 불가능한 상황 발생 가능

### 물리 퍼즐
- 타이밍/정확도 요구
- 실패 시 재시도 필요

---

**❌ 이런 게임은 만들지 않는다:**
- "아무거나 눌러도 클리어"
- "순서대로 하면 무조건 클리어"
- "실패할 수 없는 게임"
