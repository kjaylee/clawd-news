# Block Bounce - 블록 바운스

## 게임 컨셉
블록 블라스트(Block Blast)의 성공을 벤치마킹한 캐주얼 퍼즐 게임. 테트리스와 1010! 스타일을 결합하여, 블록을 배치해 라인을 완성하는 중독성 있는 퍼즐 게임.

## 핵심 메카닉
- **블록 배치**: 다양한 모양의 블록을 8x8 그리드에 드래그 앤 드롭
- **라인 클리어**: 가로/세로 라인이 완성되면 해당 라인 제거 + 점수 획득
- **콤보 시스템**: 연속 클리어 시 콤보 배율 적용
- **엔드리스 모드**: 블록을 배치할 공간이 없으면 게임 오버

## 타겟 유저
- 캐주얼 게이머 (20-50대)
- 출퇴근/대기 시간에 짧게 즐기는 유저
- 퍼즐 게임 선호 유저

## 차별점
1. **미니멀 디자인**: 깔끔한 파스텔톤 UI
2. **무료 + 광고 모델**: 저사양 기기 최적화
3. **오프라인 플레이**: 인터넷 불필요
4. **원터치 조작**: 간단한 드래그 앤 드롭

## 수익 모델
- 배너 광고 (하단)
- 리워드 광고 (추가 기회, 힌트)
- 광고 제거 IAP ($2.99)

## 기술 스택
- HTML5 Canvas
- Vanilla JavaScript (의존성 최소화)
- LocalStorage (점수 저장)
- PWA 지원 (오프라인)

## 시스템 설계

### 게임 상태
```javascript
{
  grid: [8][8],        // 0: 빈칸, 1-7: 색상 코드
  score: number,
  highScore: number,
  combo: number,
  currentBlocks: [3],  // 선택 가능한 3개 블록
  gameOver: boolean
}
```

### 블록 타입 (7종)
1. I형 (1x4)
2. L형 (3x2)
3. T형 (3x2)
4. O형 (2x2)
5. S형 (3x2)
6. 단일 (1x1)
7. 큰 사각형 (3x3)

### 점수 계산
- 블록 배치: 블록 셀 수 × 10
- 라인 클리어: 100 × 클리어 라인 수
- 콤보 보너스: 기본 점수 × 콤보 배율
