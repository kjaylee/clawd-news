## 🚀 발행 단계

### 1단계: Medium 계정 준비
1. [medium.com](https://medium.com) 로그인
2. Partner Program 가입 확인 (Settings → Monetization)
3. Stripe 연결 확인 (수익 정산용)

### 2단계: 새 스토리 작성
1. **+ 버튼** 클릭 → "New story"
2. 마크다운 파일 내용 복사-붙여넣기
3. Medium이 자동으로 포맷팅

### 3단계: 이미지 추가 ⚠️ 중요
아티클에 `[IMAGE PLACEHOLDER]` 표시된 곳에 이미지 필요:

**옵션 A: Unsplash (무료, 쉬움)**
1. [unsplash.com](https://unsplash.com) 방문
2. 관련 키워드 검색 (예: "coding", "game", "productivity")
3. 이미지 다운로드 → Medium에 드래그앤드롭

**옵션 B: 직접 스크린샷**
1. 완성된 게임 스크린샷 캡처
2. 코드 에디터 스크린샷
3. 실제 결과물 화면

**옵션 C: AI 이미지 생성**
1. DALL-E 또는 Midjourney 사용
2. 프롬프트: "Clean minimal illustration of [주제], tech blog style, flat design"

### 4단계: 메타데이터 설정
**스토리 설정 (⋯ 메뉴 → Story settings):**
- **Title & Subtitle:** 이미 작성됨
- **Featured image:** 첫 번째 이미지 자동 선택
- **Tags:** 최대 5개 (SEO 중요!)

**추천 태그:**

HTML5 게임 튜토리얼:
```
JavaScript, Web Development, Game Development, Programming, Tutorial
```

AI 생산성 가이드:
```
Artificial Intelligence, Productivity, ChatGPT, Self Improvement, Technology
```

### 5단계: Publication 제출 (선택사항, 권장)
Publications에 제출하면 더 많은 독자에게 노출됩니다.

**추천 Publications:**
- **JavaScript in Plain English** (JS 콘텐츠)
- **Better Programming** (개발 튜토리얼)
- **The Startup** (일반 테크)
- **Towards AI** (AI 콘텐츠)

**제출 방법:**
1. Publication 페이지 방문
2. "Write for us" 또는 제출 가이드라인 확인
3. 대부분 간단한 폼 작성 필요

### 6단계: 발행!
1. "Publish" 버튼 클릭
2. Metered story 선택 (Partner Program 수익화)
3. Schedule 또는 즉시 발행

## 📈 SEO & 최적화 팁

### 제목
- ✅ 숫자 사용 ("30 Minutes", "4 Hours")
- ✅ 강한 동사 ("Built", "Replaced")
- ✅ 호기심 유발 ("Here's How", "Exact Templates")
- ❌ 클릭베이트지만 내용 없는 제목 피하기

### 서브타이틀
- 첫 줄이 검색 결과에 표시됨
- 핵심 가치 즉시 전달
- 120자 이내 권장

### 내부 링크
- 다른 Medium 아티클 링크 추가
- 외부 사이트 (eastsea.monster) 링크도 OK

### 이미지
- Alt text 추가 (접근성 + SEO)
- 첫 이미지 특히 중요 (썸네일)
- 코드 스크린샷 대신 코드블록 사용

## 💰 수익화 정보

### Partner Program 수익 구조
- 멤버 읽기 시간 기반
- 박수(claps) 수가 아닌 **실제 읽기 시간**이 중요
- 월 1회 Stripe로 정산

### 수익 극대화 전략
1. **6분 읽기 시간**이 최적 (너무 짧으면 수익↓, 너무 길면 이탈↑)
2. 스토리 초반 강력한 훅으로 이탈 방지
3. 소제목으로 스캔 가능하게 구성
4. 마지막에 다른 글 링크 (독자 유지)

### 예상 수익
- 1,000 views → $2-10
- 10,000 views → $20-100
- 바이럴 시 → $100-1000+

*수익은 독자의 멤버십 여부, 읽기 시간 등에 따라 크게 변동*
