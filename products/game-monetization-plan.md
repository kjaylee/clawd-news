- **수익 분배:** 개발자에게 수익 분배 (비율 비공개, 업계 평균 50-70%)
- **가입:** 무료 계정 생성 → SDK 통합 → 게임 업로드
- **SDK:** `<script src="https://html5.api.gamedistribution.com/..."></script>`
- **광고 유형:** 인터스티셜, 배너, 보상형 비디오
- **장점:** HTML5 게임 전문, 간단한 SDK 통합
- **단점:** 트래픽 많아야 의미있는 수익
- **적용 난이도:** ⭐⭐ (쉬움)

#### 2. GameMonetize (gamemonetize.com)
- **수익 분배:** 45% (개발자) — 업계 최고 수준 투명성
- **가입:** 무료, 즉시 승인
- **SDK:** 간단한 API 통합
- **광고 유형:** 프리롤, 미드롤, 배너
- **장점:** 90% 수익 분배 (퍼블리셔 모드), 실시간 통계
- **단점:** 자체 사이트 게시 시 수익 낮을 수 있음
- **적용 난이도:** ⭐⭐ (쉬움)

            case "SDK_GAME_START": /* 게임 재개 */ break;
