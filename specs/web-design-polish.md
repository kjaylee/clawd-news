# Web Design Polish

## 목표
메인 페이지 + 게임 포털 + 사이드바 전반 UI/UX 개선

## 메인 페이지 (index.md)
1. 게임 섹션을 통합 포털 링크로 업데이트 (기존 4개 → 대표 6개 + "전체 27종 보기" 링크)
2. 외부 jay-projects 링크 → 통합 포털 내부 링크로 전환
3. 게임 수 정보 업데이트 (15종 → 27종)

## 게임 포털 (games/index.html)
1. 헤더 영역: 타이틀 + 게임 수 카운트 + 검색바
2. 필터 바: 카테고리 태그 버튼 (전체/퍼즐/아케이드/액션/리듬/시뮬/디펜스 등)
3. 그리드 카드: 이모지 프리뷰 + 제목 + 설명 + 태그 + 배지
4. 배지: "NEW" (clawd-news 신규), "외부 링크" (jay-projects)
5. 반응형: 모바일 1컬럼, 태블릿 2컬럼, 데스크톱 3컬럼
6. 다크 테마 유지 (기존 디자인과 일관)
7. 스크롤 애니메이션

## 사이드바 (default.html)
1. 게임 포털 직접 링크 추가 (현재 기획서만 있음)
2. "🎮 게임" 섹션에 "게임 포탈 (27종)" 링크 추가
3. 네비게이션 순서 정리

## 모바일 최적화
1. 햄버거 메뉴 또는 수평 스크롤 내비
2. 터치 인터렉션 최적화
3. 카드 사이즈 조정
