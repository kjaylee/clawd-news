## 🔄 핵심 루프 (Core Loop)

### Moment Loop (순간, 1-5초)
```
[행동] → [결과] → [피드백] ↻
```

### Session Loop (세션, 2-5분)
```
[시작] → [핵심플레이] → [결과] → [보상] → [다음?] ↻
```

### Day Loop (일일, 해당 시)
```
[데일리 보상] → [세션] → [진행] → [내일 동기] ↻
```

### Meta Loop (장기, 해당 시)
```
[컬렉션/진행] → [새 콘텐츠] → [목표 갱신] ↻
```
