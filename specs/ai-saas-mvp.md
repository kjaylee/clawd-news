### 상세 평가

#### 1. AI 콘텐츠 리퍼포저 ✅ **선정**
- **장점:**
  - Gemini Pro 무료로 LLM 비용 0원
  - 플랫폼별 템플릿만 만들면 됨 (코딩 간단)
  - 명확한 가치 제안: "블로그 1개 → SNS 10개"
  - 콘텐츠 마케터/1인 창업자 타겟 명확
- **단점:**
  - Repurpose.io, Castmagic 등 경쟁자 존재
  - 차별화 필요 → **한국어 특화 + 가격 경쟁력**

#### 2. AI 비디오 더빙/립싱크
- **장점:** 시장 급성장, 경쟁자 적음 (HeyGen, Synthesia 정도)
- **단점:** 립싱크 ML 모델 직접 구축 필요, 2주 불가능
- **결론:** Phase 2로 미룸

#### 3. AI 비서 구독 (미스 김 SaaS화)
- **장점:** 독보적 기술, 경쟁 없음
- **단점:** 멀티테넌시, 보안, 인프라 복잡도 극심
- **결론:** 6개월+ 프로젝트, MVP 부적합

#### 4. MLX 이미지 생성 API
- **장점:** 이미 구축됨, API 레이어만 추가
- **단점:** 
  - 180초/이미지 → 사용성 최악
  - DALL-E 3, Midjourney와 경쟁 불가
  - 가격 경쟁력 없음 (대형사 규모의 경제)
- **결론:** 탈락

## 2. 선정 MVP: ContentForge

### 한 줄 요약
> **"블로그 하나 던지면 SNS 10개 나온다"**

### 타겟 고객
- 1인 창업자 / 인플루언서
- 콘텐츠 마케터 (중소기업)
- 블로거 / 유튜버 (텍스트 콘텐츠 재활용)

### 핵심 가치 제안
1. **시간 절약**: 블로그 1개 작성 → 10개 플랫폼 콘텐츠 자동 생성
2. **플랫폼 최적화**: 각 SNS 특성에 맞게 톤/길이/해시태그 자동 조정
3. **한국어 특화**: 한국 SNS (네이버, 카카오, 티스토리) 지원

## 4. 기술 스택

### 프론트엔드
- **Next.js 14** (App Router)
- **Tailwind CSS** + **shadcn/ui**
- **Vercel** 배포 (무료 티어)

### 백엔드
- **Next.js API Routes** (서버리스)
- **Supabase** (DB + Auth, 무료 티어)
- **Gemini Pro API** (MiniPC 프록시 경유, 무료)

### 인프라
```
[사용자] → [Vercel] → [Supabase] 
                   ↓
            [MiniPC Gemini Proxy]
                   ↓
            [Gemini Pro API]
```

### 비용 구조 (월)
| 항목 | 비용 |
|------|------|
| Vercel | $0 (Hobby) |
| Supabase | $0 (Free tier) |
| Gemini API | $0 (MiniPC 경유) |
| 도메인 | $12/년 |
| **총** | **~$1/월** |
