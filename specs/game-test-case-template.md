# 게임 테스트 케이스 템플릿

## 모든 게임 공통 테스트

### 1. 초기 로드
- [ ] 페이지 3초 내 로드
- [ ] JS 콘솔 에러 없음
- [ ] 메뉴/시작 화면 정상 표시
- [ ] 모바일 뷰포트(390x844) 레이아웃 정상

### 2. 시작 플로우
- [ ] 시작 버튼 터치 반응 (44x44px 이상)
- [ ] 시작 후 게임 화면 전환 (300ms 이내)
- [ ] 튜토리얼 or 직관적 UI (3초 내 무엇을 해야 하는지 이해 가능)
- [ ] 시작 후 최소 10초 생존 가능 (입력 없이도)

### 3. 핵심 게임플레이
- [ ] 터치 입력 → 즉각 시각적 피드백 (100ms 이내)
- [ ] 게임 목표가 명확 (점수/레벨/클리어 조건)
- [ ] 난이도 곡선 존재 (쉬움→어려움 점진적)
- [ ] 최소 5분 이상 플레이 가능한 콘텐츠
- [ ] 반복 플레이 동기부여 (하이스코어/레벨/보상)

### 4. 승리/패배
- [ ] 승리 조건 명확 & 달성 가능
- [ ] 패배 조건 명확 & 공정함
- [ ] 승리 시 축하 이펙트 (최소 1초 표시)
- [ ] 패배 시 결과 화면 (점수/통계)
- [ ] 재시작 버튼 즉시 접근 가능

### 5. UI/UX
- [ ] 모든 버튼 터치 영역 44x44px 이상
- [ ] 애니메이션 최소 200ms (인간 인지 가능)
- [ ] 점수/보상 표시 1초 이상 유지
- [ ] 화면 전환 300-500ms
- [ ] 번쩍임/깜빡임 없음 (50ms 이하 효과 금지)
- [ ] 폰트 크기 읽기 편함 (최소 14px)

### 6. 사운드 (선택)
- [ ] 효과음 과하지 않음
- [ ] 음소거 옵션 존재

### 7. 저장/진행
- [ ] localStorage 저장 정상
- [ ] 새로고침 후 진행 유지
- [ ] 하이스코어 유지

### 8. 엣지 케이스
- [ ] 빠른 연타 시 크래시 없음
- [ ] 화면 회전 시 깨지지 않음
- [ ] 뒤로가기 시 정상 처리
- [ ] 장시간 방치 후 복귀 정상

## 장르별 추가 테스트

### 퍼즐 게임 (Match-3, 소트, 머지 등)
- [ ] 퍼즐 해법 존재 (풀 수 없는 레벨 없음)
- [ ] 힌트 시스템 작동
- [ ] 되돌리기(Undo) 작동
- [ ] 셔플 기능 (막힘 방지)
- [ ] 레벨 간 난이도 차이 체감
- [ ] 같은 레벨 반복 시 다른 배치

### 아케이드 게임 (스네이크, 러너, 슈터 등)
- [ ] 조작감 즉각적 (입력→반응 50ms 이내)
- [ ] 충돌 판정 공정 (억울한 죽음 없음)
- [ ] 속도 증가 곡선 적절 (갑자기 못 따라갈 정도 X)
- [ ] 파워업/아이템 효과 명확
- [ ] 콤보/연쇄 시스템 보상감

### 서바이벌/로그라이크 (뱀서류)
- [ ] 자동 공격 정상 작동
- [ ] 레벨업 선택지 밸런스 (함정 선택지 없음)
- [ ] 적 스폰 패턴 다양
- [ ] 보스/웨이브 변화 체감
- [ ] 10분 이상 생존 가능

### 아이들/타이쿤
- [ ] 오프라인 수익 계산 정확
- [ ] 업그레이드 체감 가능
- [ ] 목표/마일스톤 존재
- [ ] 프레스티지/리셋 메카닉 (있으면)

### 전략/타워디펜스
- [ ] 유닛 배치 직관적
- [ ] 적 경로 명확
- [ ] 밸런스 (너무 쉽거나 너무 어렵지 않음)
- [ ] 웨이브 간 준비 시간 충분
